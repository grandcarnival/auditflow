# PowerPoint Package

This package is the start of the AuditFlow AI PowerPoint preservation engine.

Current implementation:

- Open XML package metrics.
- Slide text extraction.
- Prior-slide template structure extraction.
- Clone-and-edit text replacement.
- Table-cell replacement with row insertion/removal.
- Chart XML cache and embedded workbook updates.
- Slide duplication with relationship remapping.
- Template layout fingerprints and semantic placeholder detection.
- Merged-cell fixture coverage.
- Image asset deduplication coverage.
- Preservation validation.

Near-term backlog:

- Layout matcher.
- Content binder.
- Rich chart fixture coverage.
- Export operation manifests.
